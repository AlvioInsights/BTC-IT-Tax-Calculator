# BTC-IT-Tax-Calculator
BTC IT Tax Calculator è un'applicazione Python locale che scansiona la blockchain di Bitcoin a partire da un indirizzo o una chiave pubblica estesa (xPub, yPub, zPub). Calcola automaticamente le simulazioni per la dichiarazione dei redditi italiana (Quadro RW per il patrimonio e Quadro RT per le plusvalenze con metodo LIFO).
